# VPN-Service

## О проекте
VPN-Service — это информационная система для предоставления пользователям защищенного VPN-подключения, управления подпиской, оплаты услуг и взаимодействия с технической поддержкой.

Проект реализован в виде микросервисной архитектуры и включает:
- клиентское приложение;
- API Gateway;
- сервис аутентификации;
- VPN Core Service;
- сервис управления серверами;
- биллинг-сервис;
- сервис мониторинга и логирования;
- PostgreSQL, Redis и VPN-инфраструктуру на базе Xray-core/VLESS.

## Основные возможности
- регистрация и авторизация пользователей;
- восстановление доступа;
- просмотр тарифов и оформление подписки;
- пополнение баланса и оплата;
- подключение и отключение от VPN;
- техническая поддержка;
- панель модератора;
- панель администратора;
- мониторинг серверов и логирование событий.

## Архитектура
Система построена на основе микросервисного подхода.  
Основные компоненты:
- **Client App** — интерфейс пользователя;
- **API Gateway** — единая точка входа;
- **Auth Service** — регистрация, вход, JWT;
- **VPN Core Service** — генерация конфигураций и работа с VPN;
- **Server Management Service** — управление VPN-узлами;
- **Billing Service** — подписки и платежи;
- **Monitoring Service** — метрики, логи, аудит;
- **PostgreSQL** — основная БД;
- **Redis** — кэш и сессии;
- **Xray-core / VLESS** — VPN-инфраструктура.
