# Документация разработчика VPN-Service

## Назначение
Данный документ предназначен для разработчиков, занимающихся разработкой, сопровождением и расширением системы VPN-Service.

Документация описывает:
- архитектуру системы;
- структуру проекта;
- описание микросервисов;
- модель данных;
- API;
- процессы развертывания и мониторинга.

## Краткое описание системы
VPN-Service — это информационная система, обеспечивающая:
- защищенное VPN-подключение;
- управление пользователями;
- управление подписками и тарифами;
- обработку платежей;
- техническую поддержку;
- администрирование системы.

Система реализована на основе микросервисной архитектуры.
