# Микросервисы

## Auth Service
### Назначение
Обеспечивает аутентификацию пользователей.

### Функции
- регистрация пользователя;
- вход в систему;
- проверка пароля;
- выдача JWT;
- восстановление доступа.

---

## VPN Core Service
### Назначение
Управляет VPN-подключениями.

### Функции
- генерация конфигураций VLESS;
- подключение пользователя к серверу;
- отключение;
- контроль соединений.

---

## Server Management Service
### Назначение
Управляет VPN-серверами.

### Функции
- мониторинг состояния серверов;
- распределение нагрузки;
- добавление новых серверов;
- обновление конфигурации.

---

## Billing Service
### Назначение
Управление финансами.

### Функции
- управление тарифами;
- оформление подписок;
- обработка платежей;
- формирование чеков.

---

## Monitoring Service
### Назначение
Контроль состояния системы.

### Функции
- сбор логов;
- мониторинг серверов;
- аудит действий;
- оповещения.
